## User
### Login
Route : `/login`  
Method : `POST`  
Headers : -   
Body : 
```
{
    "email" : "johnDoe@email.com",
    "password" : "johnDoe123"
}
```
Response : 
```
Success :
{
    "status": 200,
    "token": <TOKEN>
}

Error :
{
    "status": 400,
    "message": "Invalid Email/Password"
}
```
# 

### Register
Route : `/register`  
Method : `POST`  
Headers : -   
Body : 
```
{
    "username" : "johnDoe",
    "email" : "johnDoe@email.com",
    "password" : "johnDoe123"
}
```
Response : 
```
Success :
{
    "status": 201,
    "message": "User Registered",
    "createdUser"
}
Error :
{
    "status": 401,
    "message": "User already registered"
}
```

## Articles 
```
All of routes below need token! [ Authentication ] 
```
Headers : `TOKEN`  

### Create Articles
Route : `/articles`  
Method : `POST`  
Body : 
```
{
    title : STRING,
    content : STRING,
    tags : [],
    file : <IMAGE TYPE FILE>
}
```  
Response : 
```
Success :
{
    status : 200,
    createdArticle
}
```
# 

### Get All Articles
